Open VS Code and run Ctrl+Shift+P → "Developer: Open Extensions Folder".
Create a new folder named vscode-copy-unix-path and place the files above inside it.
Open the folder in VS Code.
Run npm init -y and install dependencies if needed.
Press F5 to launch a new Extension Development Host.
Right-click in any editor window → you should see "Copy Unix Path" in the context menu.
